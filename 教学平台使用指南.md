## 🔐 账号注册与登录

### 访问平台

在浏览器中打开平台地址：`http://localhost:3000`

### 注册新账号

#### 方式一：通过登录页面注册

1. 访问登录页面：`http://localhost:3000/login`
2. 在登录页面找到注册链接：
   - **云版本**: 点击 "Don't have an account? **Sign up for free**"
   - **企业版**: 点击 "Have an invite code? **Sign up for an account**"

#### 方式二：直接访问注册页面

直接在浏览器中访问：`http://localhost:3000/register`

### 注册信息填写

根据平台版本，需要填写以下信息：

#### 开源版 (Open Source)
- **用户名** (Username): 您的显示名称
- **邮箱** (Email): 有效的邮箱地址，用于账号验证和找回密码
- **密码** (Password): 必须满足以下安全要求：
  - 至少 8 个字符
  - 包含至少一个小写字母 (a-z)
  - 包含至少一个大写字母 (A-Z)
  - 包含至少一个数字 (0-9)
  - 包含至少一个特殊字符 (!@#$%^&* 等)
- **确认密码**: 再次输入密码进行确认

#### 企业版 (Enterprise)
除了上述信息外，还需要：
- **邀请码** (Invite Code): 由管理员提供的邀请码

### 注册流程

1. **填写注册表单**
   - 按要求填写所有必填字段
   - 确保密码符合安全要求
   - 确认密码输入一致

2. **提交注册**
   - 点击"注册"或"Sign Up"按钮
   - 系统将验证输入信息

3. **邮箱验证**（如果需要）
   - 某些配置可能需要邮箱验证
   - 检查您的邮箱收件箱
   - 点击验证链接完成验证

4. **注册成功**
   - 注册成功后可直接登录使用

### 登录账号

1. **访问登录页面**
   - 打开 `http://localhost:3000/login`

2. **输入登录信息**
   - **邮箱**: 注册时使用的邮箱地址
   - **密码**: 您设置的密码

3. **登录选项**
   - 点击"Login"按钮登录
   - 如果忘记密码，点击"Forgot password?"进行密码重置

### SSO 单点登录（如果已配置）

平台支持以下 SSO 登录方式：

- **Microsoft Azure**: 使用企业 Microsoft 账号登录
- **Google**: 使用 Google 账号登录
- **Auth0**: 通过 Auth0 服务登录
- **GitHub**: 使用 GitHub 账号登录

在登录页面点击对应的 SSO 按钮即可使用第三方账号登录。

### 密码重置

如果忘记密码：

1. 在登录页面点击"Forgot password?"
2. 输入注册时使用的邮箱地址
3. 检查邮箱中的密码重置邮件
4. 点击邮件中的重置链接
5. 设置新密码

### 账号安全建议

- **使用强密码**: 遵循密码安全要求，避免使用常见密码
- **定期更换密码**: 建议每 3-6 个月更换一次密码
- **保护账号信息**: 不要与他人分享登录凭据
- **及时退出**: 使用完毕后及时退出账号

## 🚀 平台功能介绍

### 主要功能模块

1. **智能体流程 (Chatflows)**
   - 创建和管理 AI 对话流程
   - 可视化流程设计器
   - 实时测试和调试

2. **代理流程 (Agentflows)**
   - 构建智能代理工作流
   - 多步骤任务自动化
   - 复杂逻辑处理

3. **知识库 (Document Store)**
   - 上传和管理文档
   - 文档向量化处理
   - 知识检索功能

4. **API 密钥管理**
   - 生成和管理 API 密钥
   - 控制访问权限
   - 使用统计查看

5. **凭证管理 (Credentials)**
   - 安全存储第三方服务凭证
   - 加密保护敏感信息
   - 统一凭证管理

### 快速开始

1. **登录后首次使用**
   - 系统会自动跳转到智能体流程页面
   - 可以从模板开始或创建新流程

2. **创建第一个智能体**
   - 点击"+ Create New Chatflow"
   - 选择合适的模板或从空白开始
   - 拖拽组件构建流程
   - 配置参数并测试

3. **保存和部署**
   - 保存您的智能体配置
   - 获取 API 端点进行集成
   - 监控使用情况和性能

## ❓ 常见问题

### 注册登录相关

**Q: 注册时提示"用户名已存在"怎么办？**
A: 请尝试使用不同的用户名或邮箱地址。

**Q: 没有收到验证邮件怎么办？**
A: 请检查垃圾邮件文件夹，或在登录页面点击"Resend Verification Email"重新发送。

**Q: 密码不符合要求怎么办？**
A: 确保密码至少 8 位，包含大小写字母、数字和特殊字符。

**Q: 忘记密码怎么办？**
A: 在登录页面点击"Forgot password?"，按提示重置密码。

### 功能使用相关

**Q: 如何创建第一个智能体？**
A: 登录后点击"+ Create New Chatflow"，选择模板或从空白开始创建。

**Q: 如何获取 API 密钥？**
A: 在左侧菜单中选择"API Keys"，点击"+ Add New"创建新的 API 密钥。

**Q: 如何上传文档到知识库？**
A: 访问"Document Store"页面，点击"+ Add New"上传文档。

### 技术支持

如果遇到其他问题，请：

1. 查看浏览器控制台是否有错误信息
2. 确认网络连接正常
3. 尝试清除浏览器缓存和 Cookie
4. 联系系统管理员获取帮助
